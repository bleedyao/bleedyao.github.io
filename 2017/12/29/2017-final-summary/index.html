2017 年终总结

发表于 2017-12-29 | 分类于 年终总结 | | 阅读次数:

前言

今天是 2017 年最后一个工作日，时间过得真快，一年的时间就在我的眼前这样的溜走了，这一年发生了很多事情，对我也有很大的触动，也可以这么说，这一年我成长了很多。

工作

记得去年我还在教 java 的知识，做教案，自己学习，为的是让我的学生能通过我的教学，掌握他们将来吃饭的工具。但是，这也暴露出一个问题，我利用我的所有时间总结经验，传授知识，然而我的成长点在哪里？我是在不断的放电，看不到自己的成长点在哪里。

我觉得我的青春不应该浪费在讲台，而应该投入到实际的生产实践中去。因此，今年的年初，我完成了这一职业上的转型。

来到公司以后，可以说成长速度飞快，做一款 APP 的过程中，发现问题，解决问题，苦思冥想，绞尽脑汁，这是我想要的工作状态。这一年，项目基本上一个接着一个的，比较紧密，过得比较充实，我很满意。

社交

这一年，我开始关注技术圈的一些大牛，看看他们每天在干什么。于是，今年有幸加入了 stormzhang 的知识星球，同时也跟上了消息的最前沿，很多消息，可以第一时间得知，可以提前做准备，技术方向也好，投资也好，都是和我有着切身关系的事件。因此，这是我踏入技术圈的一个新的起点，也是我成长的目标，这是这一年我最大的一个收获。当然我已经续费下一年的会员了，会继续追随，把花出去的前赚回来。

另外，年底的时候加入了任玉刚的知识星球，是为了学习技术，问一下高深的问题，增进技术。

stormzhang，郭霖，任玉刚都是和我同岁，和人家相比在技术上，我还有很大的差距，所以不努力可能以后就没有饭吃了。

在技术上，我的目标就是不要和这些大牛们拉开差距，这就是明年的目标吧。

生活

今年还有一大变化，就是我从两个人的世界回归到了一个人的世界，这是受现实所迫。经历了此事，我看清楚了自己的处境，我的想法也更加明确：作为一个在北京从小长到大的，学习成绩良好，做人谦虚不张扬，踏实本分的大男孩，应该去寻找生活在相同环境下，合得来的女孩，仅此而已。

话多无益，命数自有天定，我又何须多言，一切自有天意。

目标

2018 年，我给自己定下的目标很明确：成为 Android 高级工程师！就是这么简单，就是这么粗暴，哈哈。

坚持原创技术分享，您的支持将鼓励我继续创作！
